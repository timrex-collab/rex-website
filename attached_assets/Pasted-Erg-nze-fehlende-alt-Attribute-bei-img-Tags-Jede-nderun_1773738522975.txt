exakt lokalisiert. Kein anderer Code wird angefasst.

Regel für alle Alt-Texte:
- Beschreibend, keyword-relevant, auf Deutsch
- Format: "[Was zu sehen ist] – Rex Bedachungs GmbH Bochum"
- Maximal 100 Zeichen

Wichtige Hinweise für den Agent:
- Schaue dir jeden img-Tag vor der Änderung kurz an
- Falls der Kontext des Bildes klar vom Dateinamen oder src-Pfad 
  abweicht, passe den Alt-Text entsprechend an – die obigen Texte 
  sind Vorschläge basierend auf dem Seitenkontext
- Kein anderer Code wird verändert
- Build ausführen (npm run build) nach allen Änderungen
- Zusammenfassung: welche Alt-Texte gesetzt wurden und ob dynamische 
  Props verfügbar waren (Fix 7–9)
