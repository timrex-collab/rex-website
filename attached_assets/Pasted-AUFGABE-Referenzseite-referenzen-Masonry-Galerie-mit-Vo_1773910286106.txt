SCHRITT 4 – Lightbox implementieren
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Die Lightbox ist eine React-State-gesteuerte Komponente im selben File.
Kein externes Package.

STATE:
const [lightboxIndex, setLightboxIndex] = useState<number | null>(null);
const isOpen = lightboxIndex !== null;

LIGHTBOX-OVERLAY (wenn isOpen):
- Außen: fixed inset-0 z-50, bg-black/95, flex flex-col items-center
  justify-center, p-4, onClick={closeLightbox}

SCHLIESSEN-BUTTON:
- Oben rechts: absolute top-3 right-3
- Lucide X Icon, 22px, text-white/80, cursor-pointer, hover:text-white

HAUPTBILD-BEREICH:
- max-w-3xl w-full (onClick stopPropagation)
- Bild: w-full max-h-[60vh] object-contain rounded-md

INFO-LEISTE (unter dem Hauptbild):
- Flex, justify-between, mt-3
- Links:
  - Titel (text-white font-semibold text-base)
  - Ort · Jahr (text-white/60 text-sm mt-1)
  - Beschreibung (text-white/70 text-sm mt-2 max-w-xl leading-relaxed)
- Rechts: Navigations-Buttons ‹ und ›
  (border border-white/25 rounded-md px-4 py-2 text-white text-lg
   hover:bg-white/15, gap-2)

THUMBNAIL-LEISTE:
- mt-4, flex, gap-2, flex-wrap, justify-center
- Jedes Thumbnail: w-14 h-10 rounded object-cover cursor-pointer
  - Aktiv: ring-2 ring-white opacity-100
  - Inaktiv: opacity-40 hover:opacity-70

KEYBOARD-NAVIGATION:
useEffect(() => {
  if (!isOpen) return;
  const handler = (e: KeyboardEvent) => {
    if (e.key === "ArrowRight") navigateLightbox(1);
    if (e.key === "ArrowLeft") navigateLightbox(-1);
    if (e.key === "Escape") closeLightbox();
  };
  window.addEventListener("keydown", handler);
  return () => window.removeEventListener("keydown", handler);
}, [isOpen, lightboxIndex]);

FUNKTIONEN:
const openLightbox = (index: number) => setLightboxIndex(index);
const closeLightbox = () => setLightboxIndex(null);
const navigateLightbox = (dir: number) =>
  setLightboxIndex((prev) =>
    prev === null ? 0 : (prev + dir + projects.length) % projects.length
  );
