## AUFGABE 1: setLocation()-Fix (alle 7 Dateien)

### Globales Muster — auf JEDER der 7 Dateien anwenden:

**Schritt A:** Suche in den Imports nach `useLocation`:

```
import { useLocation } from "wouter";
```

Ersetze durch:

```
import { Link as WouterLink } from "wouter";
```

Falls `useLocation` zusammen mit anderen Imports steht (z.B. `import { useLocation } from "wouter";` als einzelne Zeile), nur diese Zeile ersetzen. Falls `Link` bereits importiert wird, füge `useLocation` einfach aus dem Import entfernen.

**Schritt B:** Suche nach der Deklaration:

```
  const [, setLocation] = useLocation();
```

Lösche diese Zeile komplett.

**Schritt C:** Suche ALLE Stellen mit diesem Muster:

```
            <Button
              size="lg"
              className="pulse-ring cta-pulse"
              onClick={() => setLocation("/kontakt")}
```

Ersetze jeweils durch:

```
            <Button
              asChild
              size="lg"
              className="pulse-ring cta-pulse"
```

Und den zugehörigen Button-Text (z.B. `>Angebot anfragen</Button>`) ersetze durch:

```
            >
              <a href="/kontakt">Angebot anfragen</a>
            </Button>
```

**Beachte:** Der Button-Text variiert pro Seite und pro CTA-Position. Hier die korrekten Texte:

- SteildachBochum: “Vor-Ort-Termin vereinbaren” (Hero) — suche auch nach weiteren setLocation-CTAs weiter unten
- BauklempnereiBochum: “Angebot anfragen” (Hero)
- Dachreparatur: “Reparatur anfragen” (Hero)
- DachwartungBochum: “Wartungsvertrag anfragen” (Hero) — prüfe ob “Wartungstermin anfragen” irgendwo steht
- AufsparrendaemmungBochum: “Beratung anfragen” (Hero)
- DachgaubeBochum: “Angebot anfragen” (Hero)
- DachfensterBochum: Alle setLocation-Stellen nach gleichem Muster fixen

**Durchsuche JEDE Datei komplett** — es gibt oft einen zweiten CTA am Seitenende mit dem gleichen setLocation-Pattern. Alle müssen gefixt werden.
