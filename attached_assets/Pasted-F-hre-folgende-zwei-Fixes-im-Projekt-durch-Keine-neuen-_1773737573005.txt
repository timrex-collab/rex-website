FIX 1 – Telefonnummer-Format vereinheitlichen (sitewide)
Problem: Die Telefonnummer erscheint im Code in verschiedenen Formaten 
(0234-583100, 0234 583100, 234 583100 etc.). Das korrekte Format ist 
+49 234 583100 – einheitlich im gesamten Projekt.

Aktion: Durchsuche ALLE Dateien im Projekt nach folgenden Varianten der 
Telefonnummer und ersetze sie durch das korrekte Format:

Suche nach (alle Varianten):
  0234-583100
  0234 583100
  (0234) 583100
  tel:0234583100
  tel:023458310
  234 583100

Ersetze durch (kontextabhängig):
  - Im sichtbaren Text (JSX): +49 234 583100
  - In tel:-Links (href): href="tel:+492345831000"
    Achtung: Prüfe, ob die Nummer tel:+49234583100 lautet – 
    falls bereits korrekt, nicht anfassen.
  - Im JSON-LD (telephone-Feld): "+49 234 583100"

Betrifft potenziell:
  client/src/pages/*.tsx (alle Seiten)
  client/src/components/*.tsx (alle Komponenten)
  client/index.html (JSON-LD im <head>)

Wichtig: Ändere das Format NUR bei der Telefonnummer 583100. 
Keine anderen Nummern anfassen.

FIX 2 – GovernmentService-Schema in Foerderung.tsx ergänzen
Datei: client/src/pages/Foerderung.tsx

Problem: Die Förderseite hat kein maschinenlesbares Signal, dass 
Rex Bedachung BAFA/KfW-Beratung anbietet. Das ist ein relevantes 
GEO-Signal für KI-Systeme.

Aktion: Füge im Helmet-Block der Seite einen zusätzlichen JSON-LD-Block 
ein (als zweites <script type="application/ld+json">-Tag, 
nach dem bestehenden Schema-Block):

{
  "@context": "https://schema.org",
  "@type": "GovernmentService",
  "name": "BAFA & KfW Förderberatung Dach Bochum",
  "provider": {
    "@id": "https://www.rex-bedachung.de/#organization"
  },
  "serviceOutput": "Zuschüsse und Förderung für energetische Dachsanierung",
  "audience": {
    "@type": "Audience",
    "audienceType": "Hauseigentümer und Vermieter in Bochum und Umgebung"
  },
  "areaServed": {
    "@type": "City",
    "name": "Bochum"
  }
}

Falls die Seite noch kein BreadcrumbList-Schema hat, füge es ebenfalls 
im Helmet-Block hinzu:
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [
    {
      "@type": "ListItem",
      "position": 1,
      "name": "Startseite",
      "item": "https://www.rex-bedachung.de/"
    },
    {
      "@type": "ListItem",
      "position": 2,
      "name": "Förderung Dach Bochum",
      "item": "https://www.rex-bedachung.de/foerderung"
    }
  ]
}
