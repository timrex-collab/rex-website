## Aufgabe 1: Google Maps Iframe hinter Consent-Klick verstecken (DSGVO)

Öffne `client/src/pages/Contact.tsx` (oder die Datei, die die Kontaktseite enthält).

Suche den Google Maps `<iframe>`. Aktuell wird er direkt beim Seitenaufruf geladen — das ist DSGVO-problematisch, weil Google-Ressourcen ohne Einwilligung geladen werden.

**Ersetze den iframe durch eine Consent-Lösung:**

1. Zeige initial einen Platzhalter-Container an derselben Stelle, wo jetzt der iframe steht:
   - Hintergrundfarbe: `#f3f4f6` (helles Grau)
   - Zentrierter Text: "Karte wird von Google Maps geladen. Mit dem Laden akzeptieren Sie die Datenschutzerklärung von Google."
   - Darunter ein Button: "Google Maps laden"
   - Der Platzhalter soll die gleiche Größe haben wie der iframe (gleiche Höhe und Breite)

2. Erst wenn der Nutzer auf "Google Maps laden" klickt, wird der iframe eingeblendet (per React State, z.B. `const [mapsConsent, setMapsConsent] = useState(false)`).

3. Styling des Platzhalters:
   - Gleiche Breite/Höhe wie der iframe
   - Abgerundete Ecken (border-radius: 8px oder passend zum bestehenden Design)
   - Der Button soll als normaler Button erkennbar sein (Hintergrund in der Primärfarbe der Seite oder #1a365d, weiße Schrift, padding, border-radius)
   - Ein kleines Map-Pin-Icon oder 📍 Emoji vor dem Button-Text ist optional aber nett

4. Ändere NICHTS am restlichen Layout der Kontaktseite.

## Aufgabe 3: Sichtbare Breadcrumb-Komponente erstellen und einbinden

### 3a: Breadcrumb-Komponente erstellen

Erstelle eine neue Datei `client/src/components/Breadcrumb.tsx`:

```tsx
import { Link } from "wouter";

interface BreadcrumbItem {
  label: string;
  href?: string;
}

interface BreadcrumbProps {
  items: BreadcrumbItem[];
}

export default function Breadcrumb({ items }: BreadcrumbProps) {
  return (
    <nav aria-label="Breadcrumb" className="w-full bg-gray-50 border-b border-gray-200">
      <div className="max-w-6xl mx-auto px-4 py-2">
        <ol className="flex flex-wrap items-center gap-1 text-sm text-gray-500">
          {items.map((item, index) => (
            <li key={index} className="flex items-center gap-1">
              {index > 0 && <span className="text-gray-300">›</span>}
              {item.href ? (
                <Link href={item.href} className="hover:text-gray-700 hover:underline">
                  {item.label}
                </Link>
              ) : (
                <span className="text-gray-700 font-medium">{item.label}</span>
              )}
            </li>
          ))}
        </ol>
      </div>
    </nav>
  );
}
```

## Aufgabe 4: Pulsierender CTA-Button (CSS-Animation)

### 4a: CSS-Animation erstellen

Füge in die globale CSS-Datei (vermutlich `client/src/index.css` oder eine ähnliche zentrale CSS-Datei) folgende Animation hinzu:

```css
@keyframes pulse-ring {
  0% {
    box-shadow: 0 0 0 0 rgba(234, 88, 12, 0.5);
  }
  70% {
    box-shadow: 0 0 0 12px rgba(234, 88, 12, 0);
  }
  100% {
    box-shadow: 0 0 0 0 rgba(234, 88, 12, 0);
  }
}

.cta-pulse {
  animation: pulse-ring 2s ease-out infinite;
}
```

Die Farbe `rgba(234, 88, 12, ...)` ist Orange — passe den RGB-Wert an, falls die CTA-Buttons auf der Seite eine andere Primärfarbe haben. Prüfe die bestehende Button-Farbe und verwende deren RGB-Werte.

### 4b: Animation auf CTA-Buttons anwenden

Suche auf ALLEN Seiten nach den primären Call-to-Action-Buttons, die typischerweise den Text "Angebot anfragen", "Jetzt anfragen", "Kostenloses Angebot" oder ähnlich tragen.

Füge diesen Buttons die CSS-Klasse `cta-pulse` hinzu. Beispiel:
- Vorher: `className="bg-orange-600 text-white px-6 py-3 rounded-lg ..."`
- Nachher: `className="bg-orange-600 text-white px-6 py-3 rounded-lg ... cta-pulse"`

**Regeln:**
- NUR auf primäre CTA-Buttons anwenden (Angebot anfragen / Jetzt anfragen / Kontakt aufnehmen), NICHT auf sekundäre Buttons, Navigation oder andere Links.
- Pro Seite maximal 2 Buttons mit der Animation (typisch: Hero-CTA und ein CTA am Seitenende). Falls eine Seite mehr als 2 solcher Buttons hat, wähle die prominentesten 2.
- Die Animation soll nur die visuelle Wirkung des Buttons verändern (pulsierender Schatten), NICHT die Größe, Form oder Farbe des Buttons selbst.
